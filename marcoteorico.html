Marco Teórico


ArcMap representa la información geográfica como una colección de capas y otros elementos en una vista de mapa.
La ventana principal de ArcMap muestra una Barra de menús, y debajo de esta se localizan las Barras de herramientas en donde se encuentran las funciones más utilizadas en ArcMap. A la derecha de la ventana se encuentran los paneles: Results, ArcToolbox, Catalog, Search, entre otros.
Por otra parte, a la izquierda se encuentra la Tabla de contenidos, en donde estarán localizadas todas las capas a utilizar.
ArcMap cuenta con dos vistas principales, la Vista de datos y la Vista de diseño y cada una de ellas nos permite interactuar con el mapa de una manera en específico. ‎En la Vista de datos, el mapa es el marco de datos. Se presenta como una ventana geográfica en la que se muestran y utilizan capas de mapa. Dentro de este marco de datos, se trabaja con información SIG presentada a través de capas utilizando coordenadas geográficas (del mundo real).
‎La Vista de diseño se utiliza para diseñar y crear un mapa para imprimir, exportar o publicar. Se pueden agregar elementos al mapa y obtener así una vista previa de cómo se verá este antes de exportarlo o imprimirlo.

Existe información ráster la cual no está relacionada con un sistema de referencia, como lo son las imágenes aéreas.
Para asociar una imagen a un sistema de coordenadas se debe realizar el proceso de georreferenciación, el cual se explicará a continuación.
Ya que se tenga abierta la aplicación ArcMap, abriremos la imagen aérea haciendo clic en el botón Add Data que se encuentra en la barra de herramientas.



Se selecciona el archivo a georreferenciar y posteriormente a través de las cartas topográficas se identificarán puntos de control utilizando coordenadas en los ejes X y Y. Para esto haremos uso de la herramienta Georeferencing ejecutándola desde la opción Customize > Toolbars.



Posteriormente teniendo la imagen seleccionada agregaremos los puntos de control.
*Nota: Si la imagen no se puede visualizar daremos clic derecho en la imagen y después en Fit To Display.
A continuación daremos clic en el ícono Add Control Points para poder indicar los puntos conocidos; esta herramienta ajusta la imagen espacialmente a través de puntos comprendidos.



El cursor se tornará de color verde, ubicaremos en donde colocaremos el punto y daremos clic derecho para seleccionar Input X y Y para ingresar las coordenadas XY. Este mismo proceso se realizará para el número de puntos que se deseen añadir.

Para poder georreferenciar sin coordenadas específicas de los puntos de control se hará uso de elementos de referencia como vías, edificaciones, puntos hidrantes, etc. Para ello se dará clic en la imagen para indicar el punto inicial y un segundo clic en el punto de la realidad. Se indicarán los puntos necesarios en distintas áreas con el mismo procedimiento para que las capas se ajusten correctamente.
Se recomienda indicar estos puntos de manera equitativa en toda la imagen.


1. Creación de shapefiles

Para la creación de ShapeFiles, se hará uso de la aplicación ArcCatalog, posteriormente del lado izquierdo se encuentra el panel llamado Catalog Tree y dentro de este se seleccionará una ubicación en donde se ubicaran los archivos.



A continuación se dará clic en el menú File > New > Shapefile. Enseguida en la ventana se ingresará el nombre del shapefile
Luego en Feature Type se dará clic en la flecha para desplegar lista y se seleccionará el tipo de capa.
Y por último en Spatial Reference se indicará el sistema de referencia, para ello se dará clic en Edit y se definirá el sistema de coordenadas del shapefile seleccionando o importando un sistema de coordenadas y una vez seleccionado se dará clic en Aceptar.
Y el nuevo shapefile aparecerá dentro de la carpeta seleccionada.


2. Edición de shapefiles

Este proceso consiste en cartografiar los elementos en una imagen.
Para la edición de los shapefiles se hará uso nuevamente de ArcMap. Primeramente se deben añadir a la Tabla de Contenidos los shapefiles creados, y luego se activará la herramienta Editor, la cual se ejecutará desde la opción Customize > Toolbars.
Después en Editor > Start editing se activará la edición en el espacio de trabajo.



Enseguida en el panel Create Features se encontrarán las capas de los shapefiles y en la parte inferior en Construction Tools se seleccionará la opción que se editará.



Al haber realizado los cambios necesarios se finalizará la edición, para ello se dirige a Editor > Stop editing y se guardarán las modificaciones.

3. Trabajo con puntos

Teniendo abierto el panel Create Features, se seleccionará en Construction Tools la herramienta Point. Nos ubicamos sobre las residencias que queremos marcar y al hacer clic se fijará el punto.
Así mismo se pueden ingresar dando clic derecho en la edificación correspondiente e ingresar las coordenadas en diferentes formatos como metros, grados decimales, MGRS, UTM, etc.


4. Trabajo con líneas

Para digitalizar líneas se selecciona la capa necesaria en el panel Create Features. Para trazar la línea se señala con un clic el punto inicial y luego se continúa indicando punto tras punto hasta completar una sección. Para finalizar la línea se hace doble clic en el último vértice.


5. Trabajando con polígonos

Para trabajar con polígonos en el panel de Create Features se selecciona la capa y se selecciona el Polygon en el Construction Tools, se marca el primer vértice y luego se continúa dibujando todo el perímetro, para finalizar igualmente se da doble clic en el último vértice.


1. Creando nuevos campos

Para la creación de nuevos campos, se dará clic derecho sobre la tabla o en la capa de la tabla de contenido y enseguida se dará clic en Open Attribute Table.
Daremos clic en el botón de Table Options de la ventana de la tabla.
Posteriormente se dará clic en Add Field, y enseguida se escribe el nombre del campo.
Seguidamente se dará clic en la flecha Tipo y se selecciona el tipo de campo y daremos clic en Aceptar.


2. Ingreso de información

Para agregar información dentro de un campo, es necesario estar en modo de edición, después abrir la tabla de atributos y de esta manera se podrán ingresar directamente los valores a un registro dentro de un campo.

3. Cálculos
a. Área, perímetro y longitud

Para calcular el área y perímetro, se dará clic derecho sobre la capa seleccionada y abriremos la Tabla de Atributos.
Luego al dar clic en Add field y asignaremos en el Nombre “Área” o “Perímetro” y en Type seleccionaremos Double y aceptamos.
Ahora, dando clic derecho en el encabezado del campo requerido (Área) se seleccionará la herramienta Calculate Geometry. Después se elige la propiedad geométrica (Área), el sistema de coordenadas y las unidades a utilizar. Para calcular el perímetro se repite el mismo proceso, pero en este caso se selecciona la propiedad Perimeter.



b. Cálculo de coordenadas XY

Nos dirigimos al panel ArcToolbox > Data Management Tools > Features en la sección de Input Features, se selecciona primeramente la capa y daremos clic en Aceptar, después se abrirá la tabla de atributos en donde se podrá ver que incrementaron los campos con los valores XY de las coordenadas.



c. Operaciones

Se pueden realizar operaciones simples y complejas, tanto cadenas de texto como operaciones matemáticas (sumas, restas, divisiones y multiplicaciones, incluyendo paréntesis).
Para ello nos dirigimos a la Tabla de atributos, damos clic derecho y seleccionamos Statistics, después hacemos clic derecho sobre el nombre del campo creado y seleccionamos Field Calculator e ingresamos la expresión.
